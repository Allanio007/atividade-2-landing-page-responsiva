# atividade2-webar - Projeto AR: Atividade 2

Autor: allanio007

Repositorio: https://github.com/Allanio007/atividade-2-landing-page-responsiva

## Descricao

Este projeto e uma aplicacao de Realidade Aumentada Web (WebAR) desenvolvida com A-Frame e AR.js.
O objetivo e visualizar e interagir com um modelo 3D no marcador Hiro usando a camera do celular.

## Tecnologias

- A-Frame
- AR.js
- GLTF/GLB

## Como utilizar

1. Prepare o marcador Hiro: https://raw.githubusercontent.com/AR-js-org/AR.js/master/data/images/HIRO.jpg
2. Acesse o link publicado no Netlify: https://shimmering-buttercream-768b50.netlify.app/
3. Permita acesso a camera no navegador.
4. Aponte para o marcador e interaja com o objeto.

Interacoes implementadas:

- Toque unico: alterna escala e executa rotacao.
- Duplo toque: eleva e retorna a posicao do objeto.

## Tutorial Passo a Passo (Para o README da entrega)

### Cena 1: Preparacao do Ambiente

- Print (sugestao): foto do marcador Hiro impresso em folha A4 ou exibido na tela de outro dispositivo.
- Legenda: Certifique-se de estar em um ambiente bem iluminado.

### Cena 2: Acesso ao Link

- Print (sugestao): tela do celular com navegador abrindo o link do projeto.
- Legenda: Ao acessar o link, o navegador solicitara permissao para uso da camera. Clique em Permitir.

### Cena 3: Magical Moment (Realidade Aumentada)

- Print (sugestao): camera apontada para o marcador com o modelo do abacate aparecendo sobre o Hiro.
- Legenda: Aponte a camera para o marcador Hiro. O modelo 3D aparecera instantaneamente.

### Cena 4: Demonstracao de Interacao

- Print (sugestao): dedo tocando na tela enquanto o objeto gira ou muda de escala.
- Legenda: Interaja com o objeto: toque unico para alternar escala e animacao de rotacao.
