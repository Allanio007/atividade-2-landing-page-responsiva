# Atividade 2 - Landing Page Responsiva

Autor: allanio007

Projeto web responsivo desenvolvido em HTML, CSS e JavaScript, pronto para entrega e publicacao no GitHub.

## O que foi implementado

- Estrutura semantica de pagina unica (landing page)
- Layout responsivo para desktop e mobile
- Menu adaptativo para telas menores
- Secao de recursos com animacao de entrada
- Formulario de contato com validacao front-end
- Estilo visual com identidade forte e moderna

## Estrutura de arquivos

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Como executar localmente

1. Abra a pasta do projeto no VS Code.
2. Execute com extensao Live Server ou abra `index.html` no navegador.

## Como publicar no GitHub

No terminal, dentro da pasta do projeto:

```bash
git init
git add .
git commit -m "feat: atividade 2 completa"
git branch -M main
git remote add origin https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
git push -u origin main
```

Substitua a URL do repositorio pela sua no GitHub.
